BG-Lime Platform
================

This package contains a static MVP for the BG-Lime homepage.

Files:
- index.html — main landing page (BG/EN toggle)
- about.html — about page
- README.md — this file

How to upload to GitHub:
1. Download and unzip this package.
2. In your repository on GitHub, click 'Add file' → 'Upload files'.
3. Drag the unzipped files (index.html, about.html, README.md) into the upload area.
4. Commit changes.
5. Vercel will auto-deploy (or press Deploy in Vercel dashboard).
